# Movie List (React Course)

This app is a movie list, it displays their title, cover and sinopses. 
This app is part of a react native course, it has an api call for the movie list using axios, vector icons,
modal, touchableopacity, usestate, stylesheet, custom component, Flat list.

The purpose of this app is to learn how to use the react native framework.
